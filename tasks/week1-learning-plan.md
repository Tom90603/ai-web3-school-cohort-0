## Day 2 — Web3 基础

### Checklist
- [ ] 安装 MetaMask 钱包（浏览器插件）
- [ ] 创建测试钱包 → 理解三者的关系：
  - **助记词（Mnemonic）**：12/24个单词，恢复钱包的唯一凭证。**泄露 = 丢币。**
  - **私钥（Private Key）**：256位随机数，签名交易的凭证。**泄露 = 丢币。**
  - **地址（Address）**：你的链上ID，公开的。给别人汇款用的。
- [ ] 切换到 Sepolia 测试网
- [ ] 从 Sepolia Faucet 领测试币
- [ ] 发送一笔测试交易
- [ ] 在区块浏览器中找到这笔交易 → 记录：
  - 交易哈希（Transaction Hash）
  - 状态（Success / Failed）
  - Gas Used
  - 区块高度（Block Number）
- [ ] 写笔记到 notes/web3-basics.md

### 关键概念（必须搞懂）

| 概念 | 一句话解释 |
|------|-----------|
| **钱包 ≠ 账号** | 钱包是"存私钥的工具"，不是传统互联网的账号密码 |
| **签名 ≠ 登录** | 签名是用私钥对一段消息加密，证明"这个地址确实是你控制的" |
| **交易 ≠ 免费** | 每笔链上操作都要付 Gas（手续费），失败也要付 |
| **Gas** | 计算资源的定价单位，Gas Price × Gas Used = 手续费 |
| **地址 ≠ 匿名** | 虽然地址是一串字符，但链上所有交易公开可查 |

### 你可能会问的问题
- 助记词和私钥到底什么关系？→ 助记词 = 私钥的人类可读编码，本质一样
- 为什么发交易要等？→ 矿工/验证者需要把交易打包进区块，网络忙的时候要排队
- 为什么失败了还要付钱？→ 计算已经执行了，只是执行结果回滚了

|------|--------|
